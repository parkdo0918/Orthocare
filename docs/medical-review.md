## 1. 시스템 개요

### 목적

OrthoCare는 근골격계 통증 환자를 대상으로:
1. **진단 분류**: 증상 패턴을 분석하여 4가지 임상 카테고리로 분류
2. **운동 처방**: 진단 결과와 환자 특성에 맞는 맞춤형 운동 프로그램 제공
3. **점진적 조정**: 사후 평가(RPE)를 통한 난이도 자동 조정

### 지원 부위

| 부위 | 상태 | 버킷 체계 |
|------|------|----------|
| **무릎** | 활성 | OA, OVR, TRM, INF |
| **어깨** | 활성 | OA, OVR, TRM, STF |
| 허리 | 예정 | - |
| 목 | 예정 | - |

### 근거 기반 접근

- **검증된 논문**: OARSI, ACR 가이드라인, 주요 정형외과/재활의학 논문
- **교육 자료**: OrthoBullets 임상 참고자료
- **전문가 검수**: 가중치 및 운동 데이터 전문가 검토 필요

## 2. 진단 분류 체계 (버킷)

### 2.1 무릎 버킷

#### OA (Osteoarthritis, 퇴행성 관절염)

| 항목 | 내용 |
|------|------|
| **정의** | 관절 연골의 퇴행성 변화로 인한 만성 통증 |
| **주요 증상** | 아침 뻣뻣함 (<30분), 계단 통증, 염발음, 활동 후 악화 |
| **감별 포인트** | 50세 이상, 양측성, 점진적 발병, 휴식 시 호전 |
| **운동 특성** | 저충격, 관절 보호, 근력 유지, 가동범위 유지 |

**진단 기준 (ACR 기반)**:
- 무릎 통증 + (50세 이상 OR 아침 뻣뻣함 <30분 OR 염발음)

#### OVR (Overuse, 과사용 증후군)

| 항목 | 내용 |
|------|------|
| **정의** | 반복적 부하로 인한 건/인대/점액낭 손상 |
| **주요 증상** | 활동 후 통증, 특정 동작 시 악화, 휴식 시 호전 |
| **감별 포인트** | 급성 외상 없음, 반복 동작 이력, 활동량 증가 |
| **예시 질환** | 슬개대퇴 증후군, 러너스니, 거위발건염 |
| **운동 특성** | 점진적 강화, 생역학 교정, 스트레칭 |

#### TRM (Trauma, 외상성)

| 항목 | 내용 |
|------|------|
| **정의** | 급성 외상/부상으로 인한 구조적 손상 |
| **주요 증상** | 급성 발병, 부종, 불안정감, 잠김 현상 |
| **감별 포인트** | 명확한 외상 이력, 급성 증상, 기능 제한 |
| **예시 질환** | ACL/PCL 손상, 반월판 파열, 측부인대 손상 |
| **운동 특성** | ROM 회복, 안정성 강화, 단계적 진행 |

#### INF (Inflammatory, 염증성)

| 항목 | 내용 |
|------|------|
| **정의** | 감염성/결정성/자가면역성 관절염 |
| **주요 증상** | 열감, 발적, 부종, 전신 증상 |
| **감별 포인트** | 안정 시 통증, 아침 뻣뻣함 >30분, 전신 증상 |
| **예시 질환** | 화농성 관절염, 통풍, 류마티스 관절염 |
| **운동 특성** | 저강도, 순환 촉진, 급성기 운동 제한 |

### 2.2 어깨 버킷

#### OA (Osteoarthritis)

- 50세 이상, crepitus, ROM 감소
- 견관절 골관절염, AC joint OA

#### OVR (Overuse)

- Painful Arc (60-120°), overhead 활동 후 악화
- 충돌증후군, 회전근개 건염

#### TRM (Trauma)

- 급성 외상, 힘 빠짐, Drop Arm 양성
- 회전근개 파열, AC joint 손상

#### STF (Stiff/Frozen, 동결견)

| 항목 | 내용 |
|------|------|
| **정의** | 관절낭의 섬유화로 인한 점진적 가동범위 제한 |
| **주요 증상** | 외회전(ER) 제한 현저, 야간통, 점진적 ROM 감소 |
| **감별 포인트** | 능동/수동 ROM 모두 제한, 3단계 진행 패턴 |
| **운동 특성** | 점진적 스트레칭, ROM 회복, 통증 범위 내 운동 |

**진행 단계**:
1. Freezing (2-9개월): 통증 우세, ROM 감소 시작
2. Frozen (4-12개월): ROM 제한 최대, 통증 감소
3. Thawing (5-24개월): ROM 점진적 회복

## 3. 증상-버킷 가중치 매핑

### 3.1 무릎 가중치 예시

| 증상 | OA | OVR | TRM | INF | 임상적 근거 |
|------|:--:|:---:|:---:|:---:|------------|
| `stiffness_morning` | **2.0** | 0 | 0 | 1.0 | OA: <30분, INF: >30분 |
| `crepitus` | **1.5** | 0.5 | 0 | 0 | 관절면 마모 징후 |
| `pain_stairs` | **2.0** | 0.5 | 0 | 0 | 슬개대퇴 압력 증가 |
| `pain_after_activity` | 1.0 | **1.5** | 0 | 0 | 과부하 징후 |
| `swelling_acute` | 0 | 0.5 | **2.0** | 1.5 | 급성 손상/염증 |
| `instability` | 0 | 0 | **2.5** | 0 | 인대 손상 시사 |
| `locking` | 0 | 0 | **2.0** | 0 | 반월판 손상 시사 |
| `warmth_redness` | 0 | 0 | 0.5 | **2.5** | 염증 징후 |
| `night_pain` | 0.5 | 0.5 | 0.5 | **1.5** | 염증/악성 의심 |

### 3.2 가중치 계산 방식

```
1. 각 증상의 버킷별 가중치 합산
2. 벡터 검색으로 유사 케이스 검색 (논문/가이드라인)
3. LLM이 두 결과 비교 후 최종 판단

예시:
- 가중치 결과: OA(15) > OVR(4) > INF(2) > TRM(0)
- 검색 결과: OA 관련 논문 5개 매칭
- 최종 판단: OA (신뢰도 85%)
```

### 3.3 검토 요청: 가중치 타당성

현재 가중치는 문헌 기반 초기 설정입니다.

## 4. 운동 처방 알고리즘

### 4.1 운동 선택 프로세스

```
1단계: 버킷 기반 필터링
  └─ 해당 버킷에 적합한 운동만 후보로 선택
     (예: OA → 저충격, 관절 보호 운동)

2단계: 난이도 필터링
  └─ 환자 신체 레벨 + NRS 기반 허용 난이도 결정
     - 신체 Lv A + NRS 0-3: low, medium, high 허용
     - 신체 Lv D + NRS 7+: low만 허용

3단계: 개인화 조정
  └─ 나이, BMI, NRS 기반 세트/반복/휴식 조정

4단계: 기능 균형 배분
  └─ Mobility → Stretching → Strengthening → Stability 균형

5단계: 루틴 순서 결정
  └─ 준비운동 → 본운동 → 마무리 순서 배치
```

### 4.2 버킷별 운동 특성

| 버킷 | 권장 운동 유형 | 주의사항 |
|------|---------------|----------|
| **OA** | 저충격 근력, ROM 유지, 유산소 | 고충격 운동 제외, 통증 범위 내 |
| **OVR** | 점진적 강화, 생역학 교정 | 원인 동작 분석, 점진적 복귀 |
| **TRM** | ROM 회복, 안정성, 고유수용감각 | 구조 손상 확인 후 진행 |
| **INF** | 저강도 가동성, 순환 촉진 | 급성기 운동 제한, 열감 시 중단 |
| **STF** | 점진적 스트레칭, ROM 회복 | 통증 유발 범위 직전까지만 |

### 4.3 운동 데이터 스키마

각 운동에 포함된 정보:

| 필드 | 설명 | 예시 |
|------|------|------|
| `name_kr/en` | 운동명 | 힐 슬라이드 / Heel Slide |
| `difficulty` | 난이도 | low / medium / high |
| `diagnosis_tags` | 적응 버킷 | ["OA", "TRM"] |
| `function_tags` | 기능 분류 | ["Mobility", "Stretching"] |
| `target_muscles` | 타겟 근육 | ["햄스트링", "대퇴사두근"] |
| `sets/reps/rest` | 처방 용량 | 2세트, 15회, 30초 |
| `description` | 수행 방법 | 누워서 발꿈치를... |

### 4.4 검토 요청: 운동 데이터

## 5. 개인화 조정 로직

### 5.1 연령 기반 조정

| 연령대 | 조정 내용 | 근거 |
|--------|----------|------|
| **65세 이상** | 세트 -1, 휴식 +15초 | 근감소증, 피로도 고려 |
| **50-64세** | 휴식 +10초 | 회복 시간 증가 |
| **40세 미만** | 기본값 유지 | - |

**고령자 추가 조정**:
- 균형/안정성 운동 우선순위 상향
- 낙상 위험 운동 제외 또는 수정

### 5.2 BMI 기반 조정

| BMI 범위 | 조정 내용 | 근거 |
|----------|----------|------|
| **30 이상** (비만) | 근력운동 세트 -1, 휴식 +15초 | 관절 부하 고려 |
| **25-29.9** (과체중) | 휴식 +5초 | 심폐 부담 고려 |
| **18.5-24.9** | 기본값 유지 | - |

**비만 추가 조정**:
- 체중 부하 운동 주의
- 가동성/스트레칭 우선순위 상향

### 5.3 통증(NRS) 기반 조정

| NRS 범위 | 조정 내용 | 임상적 의미 |
|----------|----------|------------|
| **7-10** (심함) | 세트 -1, 반복 -3회 | 고통증 환자, 저강도 위주 |
| **4-6** (중등도) | 반복 -2회 | 중등도, 고강도 제외 |
| **0-3** (경미) | 기본값 유지 | 적극적 운동 가능 |

### 5.4 신체 점수 레벨

사전 평가 4문항 (각 1-4점, 총 4-16점):

| 레벨 | 점수 | 허용 난이도 | 운동 개수 |
|------|------|------------|----------|
| **A** | 14-16 | low, medium, high | 6개 |
| **B** | 11-13 | low, medium, high | 5개 |
| **C** | 8-10 | low, medium | 4개 |
| **D** | 4-7 | low | 3개 |

### 5.5 검토 요청: 개인화 로직

**피드백 필요**:

## 6. 안전장치 (Red Flag)

### 6.1 무릎 Red Flag

| 코드 | 증상 | 조치 |
|------|------|------|
| `fever_with_joint_pain` | 발열 + 관절통 | 화농성 관절염 의심, 즉시 병원 |
| `severe_trauma` | 심한 외상 후 변형/부종 | 골절/탈구 의심, 응급 |
| `locking_unable_to_move` | 관절 잠김으로 움직임 불가 | 반월판 끼임 의심 |
| `night_pain_severe` | 심한 야간통 (안정 시 악화) | 악성 종양 배제 필요 |
| `rapid_swelling` | 2시간 내 급격한 부종 | 혈관절증 의심 |
| `numbness_weakness` | 저림/근력약화 (신경 증상) | 신경 압박 평가 필요 |

### 6.2 Red Flag 감지 시 동작

```
1. 즉시 알림: "전문의 상담이 필요합니다"
2. 운동 추천 중단
3. 진단 결과만 제공 (부분 응답)
4. 권장 조치 안내 (응급실/외래)
```

### 6.3 검토 요청: Red Flag

**피드백 필요**:
- [ ] 현재 Red Flag 목록이 충분한가?
- [ ] 누락된 위험 징후가 있는가?
- [ ] 각 Red Flag의 권장 조치가 적절한가?

## 7. 사후 평가 시스템

### 7.1 RPE 기반 평가 (3문항)

운동 완료 후 환자가 응답:

| 문항 | 척도 | 측정 목적 |
|------|------|----------|
| 운동 난이도 체감 | 1-5 | 전체 부하 적절성 |
| 근육 자극 정도 | 1-5 | 타겟팅 효과 |
| 땀 배출량 | 1-5 | 운동 강도 |

**추가 수집**:
- 운동 중 통증 (0-10)
- 건너뛴 운동 목록
- 완료 세트 수 / 총 세트 수

### 7.2 난이도 조정 로직

3세션 RPE 합계 기반 자동 조정:

| RPE 합계 | 해석 | 다음 세션 조정 |
|----------|------|---------------|
| **≤ 6** | 너무 쉬움 | 난이도 +1, 세트 +1, 반복 +2 |
| **7-11** | 적정 | 유지 |
| **≥ 12** | 너무 힘듦 | 난이도 -1, 세트 -1, 반복 -2 |

### 7.3 예외 처리

| 상황 | 처리 |
|------|------|
| 운동 중 통증 증가 (NRS +2 이상) | 다음 세션 난이도 하향 |
| 자주 건너뛴 운동 | 해당 운동 우선순위 하향 |
| 7일 이상 미운동 | 이전 기록 리셋, 초기화 |

### 7.4 검토 요청: 사후 평가

**피드백 필요**:
- [ ] RPE 3문항이 운동 부하 평가에 충분한가?
- [ ] 조정 기준 (합계 6, 12)이 적절한가?
- [ ] 통증 증가 시 조정 로직이 안전한가?

## 8. 검토 요청 사항

### 8.1 우선순위 높음

1. **가중치 검토**: 증상-버킷 가중치 매핑의 임상적 타당성
2. **Red Flag 검토**: 누락된 위험 징후 및 조치 적절성
3. **운동 데이터 검토**: 버킷별 운동 구성 및 난이도 분류

### 8.2 우선순위 중간
