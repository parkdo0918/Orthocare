# 무릎 가중치표 v1.1 (설문 기반) · storage: `data/knee/weights.json`

설문 응답을 증상 코드로 매핑한 뒤, 버킷(OA/OVR/TRM/INF)별 가중치를 합산해 진단/버킷 추론에 사용합니다. 이 표는 `data/knee/weights.json` 생성 시 단일 진실 소스로 사용하세요. 설문 수정 시 **이 문서와 JSON 파일을 함께 업데이트**합니다.

## 설문 → 증상 코드 매핑 요약
- 위치(중복 최대 2): 안쪽 `pain_medial`, 바깥 `pain_lateral`, 앞쪽 `pain_anterior`, 양측 `bilateral_symmetric`
- 발병/시작(중복): 다친 이후 `trauma_recent`, 운동 후 `after_exercise`, 서서히 `insidious`, 1개월+ `chronic`, 활동 증가 후 `overuse_recent`
- 악화 상황(중복): 오래 걷기/서기 `prolonged_standing_walking`, 계단 오르기 `stairs_up`, 계단 내리기 `stairs_down`, 쪼그림/양반다리 `deep_flexion`, 운동 후 `after_exercise`, 비틀기/회전 `twisting`, 아침 첫걸음 `stiffness_morning`, 가만히 있을 때 `rest_pain`
- 통증 느낌(중복): 걸리거나 잠김 `locking`, 뻐근·묵직 `achy_dull`, 조조강직 <30분 `stiffness_short`, 조조강직 ≥30분 `stiffness_30min_plus`, 붓고 뜨거움 `swelling_heat`, 압통 `tenderness`
- 자동 포함: 최근 외상 `trauma`, 부종/종창 `swelling`, 하행 시 악화 `stairs_down`(중복 확인)
